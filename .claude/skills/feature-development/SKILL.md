# Feature development

1. Read the feature description from the features folder
2. plan out the feature and write out the plan to features/<feature_name>\_plan.md
3. implement the feature
4. Create instructions on how to test the feature in features/<feature_name>\_how-to-test.md
5. Commit it to a <feature_name> branch
